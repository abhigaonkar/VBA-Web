📋 Application Description

The HV Pack application manages the final packaging stage of high-voltage finished materials. It handles:

  • Packing finished items into hampers
  • Multi-tube batch management (up to 8 batches)
  • Strapping material application
  • Optional return hamper management
  • Waffle board stacking
  • Handling unit creation (HUMO)
  • Label printing for shipment

📐 Data Model

Packing Transaction Structure

  • Timestamp, Operator, Process Order
  • Item Number (finished item)
  • Batch1-Batch8 (up to 8 batch numbers)
  • Measurements (length, bow)
  • Hamper Details (ID, qty, location, movement type)
  • Strapping (item, qty=24×hamper_qty)
  • Return Hamper (optional)
  • Waffle Board (optional, item 124451)

🔄 Packing Workflow

  1. Enter finished item number
  2. Enter batch numbers (duplicate check)
  3. Enter measurements per batch
  4. For each batch: Submit tube
  5. After last batch: Show hamper options
  6. Select hamper & strapping
  7. Optional: Add return hamper
  8. Optional: Add waffle boards
  9. Submit hamper
  10. SAP: COR6 + MIGO + HUMO + Label print
