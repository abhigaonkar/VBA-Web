🔥 Bake Application Schema

Columns Field Type Description
A-B Timestamp, Operator DateTime, Text Audit trail
C-F Process Order, Item, Pallet Number SAP references
G-N Batch1-Batch8 Text Up to 8 batch numbers
O-Q Measurements Decimal Length, Bow values
R-U Hamper Config Mixed Hamper ID, Qty, Location, Mov Type
V-Y Strapping Mixed Item, Qty (24×hamper_qty)

📦 HV Pack Application Schema

  • Batch Tracking: 8 batch fields (Batch1-Batch8)
  • Measurements: Length, Bow (×2 points)
  • Hamper Config: ID, Qty, Location, Movement Type
  • Strapping: Item number, Qty (auto: 24× hamper qty)
  • Optional Returns: Return item, qty, movement type
  • Optional Waffle: Item 124451, quantity

🎣 LampRod Application Schema

  • Furnace: FR11-FR14 codes
  • Production: Pallet card, process order, item
  • Metrics: Quantity produced, labor hours
  • Scrap: Up to 4 entries (reason, amount, flags)
  • Equipment: Hours allocation per furnace
  • Sand Transfer: Item, batch, qty, locations

📐 Serialized Material Schema

  • OD Measurements: OD1, OD2, OD3, OD4
  • WALL Measurements: WALL1-WALL8
  • Bow Value: Decimal
  • Exact Length: Measured length
  • Defect Code: 13 category codes
  • Material Routing: Webb/Scrap-D/Plate/Trash flags
  • Serial Number: Auto-generated per furnace
