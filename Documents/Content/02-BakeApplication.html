📋 Application Description

The Bake Application manages vacuum furnace baking operations for semi-finished materials. It supports:

  • Serialized Bake: Tracked batches with full genealogy
  • Non-Serialized Bake: Standard production runs with quantity tracking
  • Sand Transfer: Real-time material movements between furnaces
  • Hamper Packing: Final packaging for shipment
  • Quality Control: Non-conformity flagging and defect recording

📐 Data Model

Sheet1: Transaction Log

  • Furnace / Timestamp / Operator
  • Pallet Card / Process Order / Item
  • Batch1-8 / Bake Hours
  • Hamper Config / Strapping / Returns
  • SAP Posted Flag

Sheet2: Configuration

  • Online/Offline mode
  • Last used values (defaults)
  • Furnace equipment assignments
  • Material cache & specs

🔄 User Workflow

  1. Open BakeApplication.xlsm
  2. Select furnace and mode (online/offline)
  3. Enter pallet card number
  4. Enter batch numbers (1-15)
  5. Enter bake hours
  6. Select hamper & strapping
  7. Submit transaction
  8. Save locally + backup to network
  9. If online: Post COR6 to SAP
  10. Display completion
