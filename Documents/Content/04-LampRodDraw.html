🔄 Draw & Sand Transfer Workflow

Production Draw:

  1. Select furnace (FR11-14)
  2. Enter process order
  3. Enter production metrics
  4. Optional: Record scrap (0-4 entries)
  5. Submit transaction
  6. Save locally + backup
  7. If online: COR6 + MIGO posting

Sand Transfer:

  1. Click "Transfer Sand"
  2. Enter sand item & batch
  3. If online: MB52 query for stock
  4. Select source location
  5. Enter transfer quantity
  6. Confirm & execute (or queue if offline)
